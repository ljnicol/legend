# legend
Reusable legend for your visualisations.

# Development
- ```yarn install```
- ```yarn run elm make```
- ```yarn run elm-test```

## Documentation
- ```yarn run parcel examples/index.html```

# Links

How to release Elm package:
```
elm bump
git add elm.json
git commit -m "2.0.0 - description of changes"
git tag 2.0.0
git push origin master 2.0.0
elm publish
```
